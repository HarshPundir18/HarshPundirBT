A partir de 25 de Maio de 2018 vai entrar em vigor um novo Regulamento Geral de Proteção de Dados Pessoais que obriga a uma alteração no processo de tratamento de dados pessoais pelas empresas.

Solicitamos-lhe, por isso, que valide os seus dados pessoais na sua área de utilizador e confirme que poderemos utilizar os mesmos para os fins descritos.
